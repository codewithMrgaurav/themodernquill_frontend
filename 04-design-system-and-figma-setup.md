## 04 – Design System & Figma Setup

- **Goals**:
  - Create a reusable, token-based design system in Figma for all 12+ screens.
- **Tasks**:
  - Define color tokens (primary, secondary, accent, neutrals, semantic).
  - Define typography scales (H1–H6, body, labels) using Inter/Poppins.
  - Create spacing, radius, shadows, and grid system for 1440 / 768 / 375.
  - Build core components: buttons, inputs, navbars, cards, ads, SEO blocks.
- **Deliverable**:
  - Figma page `Design System` with documented tokens and core components.
